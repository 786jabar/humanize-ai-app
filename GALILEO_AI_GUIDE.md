## Part 1: Setup Galileo AI (5 minutes)

### Step 1: Go to Galileo AI Website
1. Open your browser
2. Go to: **www.galileo.ai**
3. Click **"Sign Up"** or **"Get Started"**
4. Choose email or Google login (Google is easier)
5. Click **"Continue with Google"** and log in
6. Confirm email if needed

### Step 2: Create a New Project
1. After login, click **"+ New Design"** or **"Create Project"**
2. Choose template or start blank
3. Name your project: **"Humanize.AI"**
4. Click **"Create"**

## Part 2: Generate Design with AI (10 minutes)

### Step 3: Write Your Design Prompt

This is the most important step! Copy and paste this prompt into Galileo AI:

```
Create a professional UI design for an AI text transformation app called "Humanize.AI" with these features:

DESIGN REQUIREMENTS:
- Color scheme: Purple (#8B5CF6) to Indigo (#6366F1) gradient
- Dark and light modes
- Modern, clean, professional look

PAGE LAYOUT:
1. Header:
   - Logo "Humanize.AI" on the left
   - Navigation menu in center (Home, Academic, Features)
   - Theme toggle (dark/light) on right
   - Profile icon on far right
   - Height: 64px with subtle border bottom

2. Main Content Area (Two Column):
   - LEFT COLUMN (60%):
     * Title: "Enter Your Text"
     * Large textarea for input (placeholder: "Paste your AI-generated text here...")
     * Character counter below textarea
     * Input height should be ~300px
   
   - RIGHT COLUMN (40%):
     * Settings Card with:
       - Style dropdown (Academic, Formal, Casual, Creative, Technical, Conversational)
       - Emotion dropdown (Neutral, Confident, Persuasive, etc.)
       - Tone dropdown
       - Vocabulary level slider
       - Formality slider (0-100)
       - Complexity slider (0-100)
     * Large "Humanize Text" button (gradient purple to indigo)
     * Output area below button showing transformed text
     * Copy and Export buttons on output

3. Advanced Tools Section (Tabbed):
   - Tabs: Transform, Summarize, Score Text, Citations
   - Each tab has relevant inputs and outputs
   - Professional styling with cards

4. Footer:
   - Copyright text
   - Links section
   - Light design

STYLING:
- Font: Modern sans-serif (Inter or similar)
- Button style: Gradient purple to indigo with white text
- Cards: Clean white (light mode) or dark gray (dark mode)
- Spacing: Generous, professional padding
- Border radius: 8px for buttons, 12px for cards
- Shadows: Subtle, not overwhelming
- Mobile responsive: Stack single column on mobile

Create 3 versions:
1. Desktop layout (1440px)
2. Tablet layout (768px)
3. Mobile layout (375px)

Make it look professional, modern, and production-ready.
```

### Step 4: Paste Prompt into Galileo

1. Click **"Generate"** or **"Create with AI"** button
2. Find the text input area
3. **Copy the prompt above** (the whole thing in the code block)
4. **Paste it** into the Galileo input
5. Click **"Generate"** or **"Create Design"**

### Step 5: Wait for Generation
- Galileo will generate your design (takes 30 seconds to 2 minutes)
- You'll see a preview of the design being created
- Once done, you'll see the complete UI design

## Part 4: Export to Figma (10 minutes)

### Step 9: Export Option 1 - Direct to Figma (EASIEST)

1. In Galileo AI, look for **"Export"** button
2. Click **"Export to Figma"** (if available)
3. This opens Figma and imports the design directly
4. Choose where to save in your Figma workspace
5. Done! Your design is now in Figma

### Step 10: Export Option 2 - Download & Import

If direct export isn't available:

1. Click **"Download"** or **"Export"**
2. Choose format: **PNG** (for preview) or **SVG** (for importing)
3. Save file to your computer
4. Open Figma.com
5. Create new file: **"Humanize.AI - Galileo"**
6. Drag and drop the downloaded file into Figma
7. Figma will import the design elements

### Step 11: Save Your Figma Design

1. Figma auto-saves, but check top-left corner shows: ‚úÖ Saved
2. Share link: Click **"Share"** button in top-right
3. Get link and save it somewhere
4. Share with team/developers for feedback

---

## Part 5: Refine in Figma (15 minutes)

### Step 12: Organize Figma Design

Now in Figma, organize your design:

1. **Clean up layers:**
   - Group related elements
   - Name layers clearly
   - Delete unused elements

2. **Add color styles:**
   - Select purple shape
   - Right-click ‚Üí "Create color style"
   - Name: "Brand/Purple"
   - Repeat for all colors

3. **Add text styles:**
   - Select heading text
   - Right-click ‚Üí "Create text style"
   - Name: "Heading 1"
   - Repeat for all text sizes

### Step 13: Create Components (Optional but Recommended)

1. Select a button
2. Right-click ‚Üí **"Create component"**
3. Name: **"Button/Primary"**
4. Repeat for: Input fields, cards, etc.
5. This makes your design reusable

### Step 14: Add Variants for Interactions

For each component:
1. Right-click ‚Üí **"Add variant"**
2. Create states: Normal, Hover, Active, Disabled
3. This shows developers all possible states

### Step 15: Create Prototype (Optional)

Link pages together:
1. Click **"Prototype"** tab (right panel)
2. Click component and drag to another frame
3. Add interactions (clicks, hovers, etc.)
4. Preview with play button

## Part 6: Share & Export (5 minutes)

### Step 16: Share Your Design

1. Top-right: Click **"Share"**
2. **"Create a link"**
3. Change permission: **"Can view"**
4. Copy link and share with:
   - Team members
   - Developers
   - Stakeholders

### Step 17: Create Design Spec Document

1. In Figma, click **"Inspect"** mode
2. Click on elements to see:
   - Size (width, height)
   - Colors (hex codes)
   - Fonts (size, weight, line height)
   - Spacing (padding, margins)
3. Screenshot or take notes
4. Send to developers

### Step 18: Export for Developers

Individual exports:
1. Right-click component
2. **"Export"** ‚Üí Choose format (PNG, SVG)
3. Use for CSS reference or assets

Or export artboard:
1. Select frame
2. Right-click ‚Üí **"Export"**
3. Choose 1x, 2x, 3x resolution
4. Download

## ‚ùå Common Issues & Solutions

### Issue: "Galileo AI doesn't understand my prompt"
**Solution:** 
- Use simpler language
- Break into smaller prompts
- Regenerate with slightly different wording

### Issue: "Design doesn't match my app colors"
**Solution:**
- Edit colors in Figma after export
- Use my DESIGN_GUIDE.md hex codes
- Right-click colors ‚Üí "Edit color"

### Issue: "Missing elements or sections"
**Solution:**
- Request new generation with more specific prompt
- Or manually add missing elements in Figma
- Use screenshots of your current app as reference

### Issue: "Mobile version doesn't look good"
**Solution:**
- Regenerate with focus on mobile
- Add to prompt: "Optimize for mobile with large touch targets"
- Manually adjust in Figma if needed
